# Envio de mensagem no WhatsApp com Python

Script para envio automático de mensagens pelo WhatsApp Web utilizando `pywhatkit`.

## Instalação

Instale a biblioteca:

```
pip install pywhatkit
```

## Uso

Execute o script com o número no formato válido e a mensagem desejada.

## Observações

* É necessário estar logado no WhatsApp Web
* Utiliza o navegador padrão
* A aba pode ser fechada automaticamente após o envio
